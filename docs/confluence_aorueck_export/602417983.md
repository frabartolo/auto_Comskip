#   
PERSON_78 R+V hat Testverfahren definiert, die neben der Prüfung angemessener Umsetzung funktionaler auch die nicht-funktionaler Anforderungen (z.B. Performance, Handhabbarkeit, Wartbarkeit) berücksichtigen:

-   Werden vor Produktivsetzung über alle Anforderungen und alle Systeme angemessene **funktionale** Tests durchgeführt?
-   Werden vor Produktivsetzung über alle Systeme angemessene **nicht-funktionale** Tests durchgeführt? 

# Vorgehen für PMs/POs

1.  Zusammen mit einem Teammember, der einen guten Blick auf die Tests hat, die Tabelle unten ausfüllen
2.   Für die noch offenen ToDos entsprechende UserStories unter Jira7aa89eb9-99a0-3d78-a855-6fe03367d635ARUECK-608 anlegen und für PI24.3 einplanen und umsetzen.
