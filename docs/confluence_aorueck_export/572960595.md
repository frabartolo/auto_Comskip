## Bewertung von KI-PERSON_54 der SiSt [PERSON_55 Intelligenz (KI) (Grundsätze)](https://confluence.ruv.de/x/xNoBHg) müssen KI-Systeme nach den Kriterien 

-   Nachvollziehbarkeit 
-   Erklärbarkeit
-   Vertrauenswürdigkeit 

eingestuft werden, damit das Risiko des KI-Systems bestimmt werden und die Konsequenzen aus der Einstufung gezogen werden können.

Dazu gibt es ein Dokument [vde-spec-90012-v1-0--en--data.pdf](https://www.vde.com/resource/blob/2177870/a24b13db01773747e6b7bba4ce20ea60/vde-spec-90012-v1-0--en--data.pdf), dass diese Kriterien messbar macht aus: [Der EDSA hat eine Checkliste zur Prüfung von KI sowie Transparenzmaßnahmen entwickeln lassen.](https://confluence.ruv.de/x/88d8J). 

Inzwischen gibt es eine Audit-Checkliste zur Einstufung von KI-Systemen: [ai-auditing\_checklist-for-ai-auditing-scores\_edpb-spe-programme\_en.pdf (europa.eu)](https://www.edpb.europa.eu/system/files/2024-06/ai-auditing_checklist-for-ai-auditing-scores_edpb-spe-programme_en.pdf)

Zusätzlich gibt es einen strukturierten Prompt für R+V GPT, der eine erste Einschätzung zur Risikoklassifizierung gemäß KI-VO ermöglicht: [KI-Verordnung ganz einfach: Wie der R+V GPT-Prompt bei der Risikoklassifizierung hilft](https://org.ruv.de/sites/00000220/ZI-Blog/Lists/Beitraege/Post.aspx?ID=919)

PERSON_56 soll die Anwendung der Messmethoden auf den ART PERSON_57 beinhalten.
