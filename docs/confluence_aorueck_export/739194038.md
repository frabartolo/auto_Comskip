### Wie können wir PERSON_176 Tests regulatorik-konform bei Verwendung der CAT dokumentieren?

PERSON_380 von PERSON_176 an CAT geplant – unklar ist: wann.

Ansonsten ist es aus regulatorischer Sicht völlig ausreichend, wenn im CAT-pdf ein Link auf die zugehörigen Abnahmetestfälle in ZephyrScale auftaucht – und die Tests dort sauber gepflegt sind.
